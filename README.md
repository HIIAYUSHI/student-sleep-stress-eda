# Student Sleep and Stress Survey Analysis

## Objective
To analyze the relationship between sleep habits, stress levels, and academic performance among students.

## Dataset
Survey responses containing sleep patterns, stress indicators (PSS-10), and academic performance.

## Tools Used
Python  
Pandas  
Matplotlib  
Seaborn  

## Analysis Performed
- Data cleaning
- Feature engineering
- Stress score calculation (PSS-10)
- Exploratory Data Analysis
- Correlation analysis

## Key Insights

• Most students report sleeping between 6–7 hours per night.

• A significant proportion of students experience moderate stress.

• Sleep duration shows a weak negative relationship with stress levels.

• Students reporting better sleep quality tend to sleep longer.

• CGPA does not show a strong relationship with sleep duration.
